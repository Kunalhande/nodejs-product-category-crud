# NodeJS Product & Category CRUD Application

A simple full-stack CRUD application built using **Node.js**, **Express**, **MySQL (RDBMS)** and basic **HTML/CSS**.

This project includes:

* Category Master with CRUD operations
* Product Master with CRUD operations
* Products linked to categories using foreign key
* Server-side pagination for product listing

Run the following SQL commands in MySQL:

```sql
CREATE DATABASE nimap_test;
USE nimap_test;

CREATE TABLE Category (
  CategoryId INT AUTO_INCREMENT PRIMARY KEY,
  CategoryName VARCHAR(100)
);

CREATE TABLE Product (
  ProductId INT AUTO_INCREMENT PRIMARY KEY,
  ProductName VARCHAR(100),
  CategoryId INT,
  FOREIGN KEY (CategoryId) REFERENCES Category(CategoryId)
);
```

## ğŸ“– Pagination Logic

Products are fetched based on page size and page number.

Formula used:

```

```

Example:

```sql
LIMIT 10 OFFSET 20
```
